# 🔐 AuthPro - Responsive Login & Registration UI

A modern and responsive authentication interface built using **HTML5, CSS3, JavaScript, Bootstrap 5, and Bootstrap Icons**.

This project was developed as part of **ApexPlanet Internship Task 2**, focusing on responsive frontend development, Bootstrap mastery, JavaScript form validation, and interactive UI design.

---

## 🌐 Live Demo

🔗https://login-registration-ui-chi.vercel.app/

## 🚀 Live Features

### 🔑 Login Page

* Responsive Login Form
* Email Validation
* Password Validation (Minimum 8 Characters)
* Show/Hide Password Toggle
* Bootstrap Alerts for Success & Error Messages
* Modern Glassmorphism UI

### 📝 Registration Page

* Full Name Validation
* Email Validation
* Password Validation
* Confirm Password Validation
* Real-Time Password Match Check

### ⚡ Dummy AJAX Email Check

The application simulates checking whether an email already exists using a predefined JavaScript array.

Example existing emails:

```javascript
const existingEmails = [
  "admin@gmail.com",
  "test@gmail.com",
  "user@gmail.com"
];
```

Features:

* Email Availability Check
* Instant Feedback
* Real-Time Validation
